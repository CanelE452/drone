## Phase 1~4 코드 구현 (Mock 우선, B 경로)

설계 결정 (사용자 지침): mock은 Olympe eDSL 재현 금지, `AnafiController` 인터페이스 수준에서만(sleep+상태변수). mock 텔레메트리에 ±10% 가짜 drift → 명령값/실측값 병기 UI를 실기체 전에 검증. RealController(Olympe 호출부)는 최대한 얇게.

### 생성 파일 (`anafi-control/`)
- `config.py` — DRONE_IP/CONTROLLER 환경변수 분기, 안전 가드(MIN 10cm/MAX 100cm/2.5m/배터리 20%).
- `drone/interface.py` — `AnafiController` 추상 인터페이스 (사용자 직관 좌표; NED 부호 반전은 real에서만).
- `drone/telemetry.py` — `Telemetry` dataclass (상태 + 누적 추정위치 + last_cmd/last_actual).
- `drone/mock.py` — sleep+상태변수, `_drift()`로 ±10% 오차. hovering 게이트/배터리 가드 포함.
- `drone/real.py` — Olympe 호출부. olympe import는 클래스 선택 시에만(lazy). moveBy NED 변환, expectation 체이닝. 누적변위는 명령값 누적 + `moveByEnd` 보정 TODO 주석.
- `server/app.py` — FastAPI+WebSocket, 명령 큐 단일 워커 직렬화, 안전 가드(guard_move), telemetry 10Hz broadcast. emergency/land_now는 큐 우회 즉시 실행 + 큐 drain.
- `server/static/index.html` — vanilla JS UI. 방향패드, 스텝선택, 명령/실측/오차 표, 긴급착륙(즉시)/모터컷오프(1초 hold-to-confirm), WASD/RF/QE/Space 키바인딩, busy 시 버튼 비활성화.
- `scripts/test_mock_controller.py`, `README.md`.
